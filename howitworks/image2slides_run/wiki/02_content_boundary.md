# Content Boundary

## grep_required

The following content must be copied from the supplied knowledge base or preserved as source-locked figures:

- RNA-seq count matrix size: 21,975 genes and 6 samples.
- Group labels: saline samples S1, S2, S3 and GlcN/E samples G1, G2, G3.
- All numeric descriptive statistics, including zero-count, detected-gene, library-size, mean, median, ratio, maximum, mode, SD, CV, P99, skewness, kurtosis, Pearson correlation, PCA variance, IQR, DESeq2, and GSEA values.
- Figures copied into `wiki/sources/media/` are source-locked and must not be redrawn or numerically altered.

## generation_allowed

- Visual hierarchy, whitespace, accent bars, framing, and non-factual slide pacing.
- Short teaching transitions that do not add or change factual claims.
- Background style, layout balance, and non-data decorative structure.

## visible_text_policy

Style, page count, scene, purpose, plugin name, workflow labels, validation labels, and source file names are control-plane metadata. They must not appear as visible slide text.
